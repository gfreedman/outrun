# sprites/assets/ — Game-Ready PNG Sprite Sheets

These are the **committed, game-ready sprite sheets** consumed by `src/sprites.ts`
and loaded by `src/preloader.ts` at runtime. Do not edit these files by hand —
they are generated by the scripts in `sprites/asset_build_scripts/`.

## How sprites.ts uses these files

`src/sprites.ts` defines pixel rectangles (`x, y, w, h`) for every sprite
within its sheet. Example:

```typescript
export const PALM_T1_STRAIGHT = { sheet: 'palm', x: 0, y: 0, w: 128, h: 256 };
```

`src/preloader.ts` loads each sheet PNG into an `HTMLImageElement`.
`renderer.ts` calls `ctx.drawImage(sheet, sx, sy, sw, sh, dx, dy, dw, dh)`
using those rectangles, scaled by the perspective scale at the sprite's
world Z position.

## Regenerating

To regenerate all sprite sheets from source art:

```bash
python3 sprites/asset_build_scripts/build_all.py
```

See `sprites/README.md` for the full pipeline.
